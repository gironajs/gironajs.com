# GironaJS

gironajs.com open source application built using Next.js 13 and TailwindCSS.

> **Warning**
> This app is a work in progress.

## Running Locally

1. Install dependencies using yarn:

```sh
yarn install
```

2. Start the development server:

```sh
yarn run dev
```
